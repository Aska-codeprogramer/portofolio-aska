HTML adalah singkatan dari HyperText Markup Language, bahasa markup standar yang digunakan untuk membuat struktur dan konten halaman web. HTML digunakan untuk mendefinisikan elemen elemen seperti teks, gambar, video, tautan, dan berbagai elemen lain yang terlihat oleh pengguna di browser. analoginya html itu seperti kerangka manusia.

CSS adalah singkatan dari Cascading Style Sheets, sebuah bahasa yang digunakan untuk mengatur tampilan visual halaman web, seperti warna, font, layout. CSS memungkinkan web designer untuk memisahkan struktur konten (yang di buat dengan html )dari tampilan visualnya. Analoginya css itu seperti pakaian yang di gunakan sama manusia.

JavaScript adalah bahasa pemrograman tingkat tinggi yang digunakan terutama untuk mengembangkan antarmuka web yang interaktif dan dinamis. Bahasa ini memungkinkan para pengembang untuk menambahkan fitur seperti animasi, interaksi dengan API. analoginya javascript itu seperti organ tubuh dalam manusia, agar web interaktif.

GitHub itu untuk platform berbasis web yang digunakan untuk hosting dan pengelolaan kode sumber menggunakan sistem kontrol versi Git. Platform ini memungkinkan pengembang untuk menyimpan, melacak, dan mengelola proyek perangkat lunak. GitHub juga menyediakan berbagai fitur untuk tinjauan kode, dan manajemen proyek.
